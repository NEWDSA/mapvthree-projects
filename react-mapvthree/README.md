## 在 react 项目中使用 mapv-three

### 安装
```
npm install mapv-three
```

### 使用
mapv-three库是一个独立的三维渲染引擎库，可以基于百度地图或单独创建三维孪生场景。
* 如果不使用百度地图可以直接import整个库或指定的模块文件，使用dom作为容器参数创建场景。

* 如果使用百度地图需要先引入百度地图jsapi，先创建地图实例，以地图实例作为容器参数创建场景。
